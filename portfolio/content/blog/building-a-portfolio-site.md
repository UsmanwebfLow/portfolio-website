# Building a Portfolio Site

A portfolio is more than a collection of links—it's a reflection of how you think and build.

## Goals

- Keep it simple and fast
- Showcase projects clearly
- Make writing and updating easy

## Stack

This site uses semantic HTML, modern CSS, and a dash of JavaScript for interactivity. Posts are written in Markdown and rendered client-side.

## Lessons learned

- Lean into CSS variables for theming
- Prefer small, composable components
- Start with content, then style it
